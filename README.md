【小冯Sir】简易安卓注册登录App
# 介绍
基于安卓API 28开发的简易的注册登录app
这是我的安卓开发课设，写的比较简陋实现了简易的注册登录功能，并没有检查用户的输入，以及注册的时候是否有已存在在账户。
并且密码是明文保存的，并未对密码进行加密。
大家看看就好。
# 软件架构说明
基于安卓API 28进行开发<br />
使用了SQLite数据库对数据进行保存<br />

# 开发环境：
Abdroid SDK最低要求版本：API 28<br />
Android Studio版本：Jellyfish | 2023.3.1<br />
Gradle 版本: 8.6<br />
